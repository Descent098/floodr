There are 3 options for installation.

## Binary Release

<span>The easiest way to get `floodr` is to download a pre-built binary from the [latest release](https://github.com/descent098/floodr/releases/latest) page.</span>

<ArchitectureHint/>

## Cargo Install

If you have [Rust](https://rustup.rs/) available on your machine you can install with [cargo](https://doc.rust-lang.org/cargo/getting-started/installation.html):

```bash 
cargo install floodr
floodr <filename>.yaml
```

## Build from source

Lastly, you can download the source code and compile it:

<Tabs>
    <TabItem label="Install the Binary Globally">
  <Steps>
    <ol>
        <li> 
        Clone the repo:

        ```bash
        git clone git@github.com:descent098/floodr.git
        ```
        </li>
        <li> 
        Change into the new directory:

        ```bash
        cd floodr
        ```
        </li>

        <li>
        Build the binary:

        ```bash
        cargo install --path .
        ```
        </li>

        <li>
        Run anywhere:

        ```bash
        floodr <filename>.yaml
        ```
        </li>
    </ol>
    </Steps>
    <Aside type={"caution"}>
        This only works if you have cargo in your `PATH` variable (meaning you can run `cargo` from your command line directly). Cargo will typically copy the binary to `$HOME/.cargo/bin`

        If you want to install the binary somewhere custom use: 

        ```bash
        cargo install --path . --root /custom/location
        ```
    </Aside>
  </TabItem>
  <TabItem label="Build the Binary Normally">
  
    <Steps>
    <ol>
        <li> 
        Clone the repo:

        ```bash
        git clone git@github.com:descent098/floodr.git
        ```
        </li>
        <li> 
        Change into the new directory:

        ```bash
        cd floodr
        ```
        </li>

        <li>
        Build the binary:

        ```bash
        cargo build --release
        ```
        </li>

        <li>
        Run from the folder:

        ```bash
        ./target/release/floodr <filename>.yaml --stats
        ```
        </li>
    </ol>
    </Steps>

  </TabItem>
</Tabs>
